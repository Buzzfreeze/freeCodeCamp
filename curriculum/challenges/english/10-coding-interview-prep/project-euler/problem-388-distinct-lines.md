# --description--

จากจุดเริ่มต้น $O(0, 0, 0)$ เส้นทั้งหมดจะถูกลากไปยังจุดจุดข้อมูลอื่นๆ   
ให้ $D(N)$ เป็นจำนวนเส้นที่แตกต่างกัน

ให้หา $D({10}^{10})$ โดยตอบเป็นเลขเก้าหลักแรก และเก้าหลักสุดท้าย มาต่อกัน
