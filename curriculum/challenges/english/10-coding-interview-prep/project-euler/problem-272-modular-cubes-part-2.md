# --description--

positive number $n$ กำหนด $C(n)$ เป็นจำนวน integers $x$ สำหรับ $1 < x < n$ และ $x^3 \equiv 1\bmod n$.

เมื่อ $n = 91$, มีค่าบวก 8 จำนวนสำหรับ $x$ เป็น: 9, 16, 22, 29, 53, 74, 79, 81 ดัวนั้น $C(91) = 8$.
