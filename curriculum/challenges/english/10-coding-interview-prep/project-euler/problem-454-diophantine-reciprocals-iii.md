# --description--

ในสมการต่อไปนี้ $x$, $y$ และ $n$ เป็นจำนวนเต็มบวก

$$\frac{1}{x} + \frac{1}{y} = \frac{1}{n}$$

สำหรับขีดจำกัด $L$ เรากำหนด $F(L)$ เป็นจำนวนของโซลูชันที่ตรงตาม $x &lt; y ≤ L$

สามารถยืนยันได้ว่า $F(15) = 4$ และ $F(1000) = 1069$

หา $F({10}^{12})$.
